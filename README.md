# ChronoSync - World Clock & Timezone Converter

A professional world clock app with live analog/digital clocks, timezone converter, and meeting planner.

## Features

- **Live Clocks Dashboard** — Analog SVG + digital clocks for multiple timezones, updating every second
- **55+ Cities** — Add/remove cities with search, persisted in localStorage
- **Timezone Converter** — Convert times between any two timezones with swap functionality
- **Meeting Planner** — Visual business hours overlap grid for scheduling across timezones
- **Day/Night Indicators** — Sun/Moon icons showing local daylight status

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS v4
- shadcn/ui components
- Lucide React icons
- Framer Motion animations
